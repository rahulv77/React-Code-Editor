# React-Code-Editor
# React + Vite

An in-browser code editor made with React and Chakra-UI library
##Live Website
[react](https://reactappcodeeditor.netlify.app/)
## Run Locally

```bash
# install dependencies

npm install

# start the dev server

npm run dev
```
